Academic Projects

Car Park Security Interface

In my first year college programming module I produced my first GUI application with Python.
It tracks the registration plates of cars in the car park to ensure security and displays saved reg plates.
The user can add or remove reg plates from the system as well as search for the ones recored in the system.

Staff Payment System

In my first year university software development module I developed a wage calculation system in Java.
The user can create objects based on staff members and assign them their hourly wage and their hours worked.
The system then calculates the staff members wages and displays them in a sorted list alongside total stats of the company.

Group Project Stock Tracking Mobile App

In my first year university group project module we developed a stock tracking app in React Native.
It informs the user of changes to stock values and educates them on stock sharing and trading strategies.
Additionally, users can save search for and save stocks to see on their front page for quicker access.
